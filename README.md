# SharePlate Backend

**SharePlate** is a community food sharing platform where users can donate surplus food and others can request it. This backend is built using **Node.js**, **Express**, **MongoDB**, and **Firebase Authentication**.

## Features
- Add, update, delete, and fetch food items.
- Submit, view, and manage food requests.
- Protected routes using Firebase Auth.
- CORS enabled for local development and Netlify deployment.
